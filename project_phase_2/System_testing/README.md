# System Testing

ไฟล์นี้อธิบายภาพรวมการทดสอบระดับระบบ (System Testing) ของ ระบบ Online shopping system ชื่อ Medusa Store

---

## 1. วัตถุประสงค์ของ System Testing

การทดสอบระดับระบบมีเป้าหมายเพื่อ:
- ตรวจสอบการทำงานร่วมกันของแต่ละโมดูลในระบบว่า **เชื่อมต่อกันถูกต้อง**
- ตรวจสอบ **กรณีการใช้งานจริง (End-to-End Scenarios)** ตั้งแต่ต้นจนจบ

---

## 2. ขอบเขตของการทดสอบ (Test Scope)

ใน System Testing นี้ ครอบคลุมฟีเจอร์หลัก ได้แก่ 

- การจัดการลูกค้า (สมัครสมาชิก, ล็อกอิน, ล็อกเอาต์)
- การจัดการคำสั่งซื้อ (สร้าง, เปลี่ยนสถานะ, ดูรายละเอียด)
- การจัดการสินค้า (สร้าง, ลบ, แก้ไข)
- การซื้อสินค้า (ตะกร้าสินค้า, โปรโมชัน, ที่อยู่จัดส่ง, วิธีการจัดส่ง, วิธีการชำระเงิน)
- การสร้างโปรโมชัน
- การจัดการแคมเปญ (สร้าง, ผูกแคมเปญกับโปรโมชัน, แก้ไขวันที่แคมเปญ)
